Notes for Designer —

Scope: 2 static pages. No JS framework needed.

Pages:

  • Home — nav, hero (name + tagline + 2 download buttons + hero phone), screenshots section, bottom CTA, footer.
  • Privacy — nav, page title, long-form policy body, footer.

Download buttons: Google Play + Apple App Store. Use official store badges if possible.

Assets available: img/icon.png, img/home.png, img/chess-puzzle.png, img/rehearsal.png, img/games.png.

Open creative decisions: color palette, typography, motion, layout rhythm, screenshot presentation (grid vs scroll vs stack), background treatment.

Constraints: mobile-responsive. Privacy page should read comfortably (max-width ~640px on body text).
